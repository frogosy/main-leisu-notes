# main-leisu-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以笔记、工具、信息展示等形式存在，彼此之间没有强依赖关系，可独立访问。

## 项目简介

`main-leisu-notes` 是一个以 HTML 页面为主的归档仓库。每个页面通常包含一个独立的主题或功能，适合用作快速参考、轻量工具或静态展示。仓库本身不绑定任何特定域名或网站，仅作为页面存放和版本管理的载体。

## 目录说明

```
.
├── index.html          # 仓库首页（可选，用于导航或说明）
├── page-xxx.html       # 单个 HTML 页面
├── assets/             # 页面依赖的静态资源（如 CSS、JS、图片等）
│   ├── style.css
│   └── script.js
└── README.md           # 本文件
```

- 根目录下的 HTML 文件均为独立页面，可直接在浏览器中打开。
- `assets/` 目录用于存放页面引用的公共资源，避免重复上传。
- 每个页面的具体用途和内容请参见页面内部说明或注释。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务或数据库。
- 页面之间无跨域或数据共享需求，可直接通过文件系统或静态服务器访问。
- 归档内容可能不定期更新或清理，请以仓库最新提交为准。
